# IP Soft Core Defect Dataset

This repository contains the dataset used in our empirical study of soft IP core defects.

## Files
- `OpenTitan_issue.xlsx`: defect records collected from OpenTitan issues
- `OpenTitan_pr.xlsx`: defect-related pull requests collected from OpenTitan
- `PULPissimo_issue.xlsx`: defect records collected from PULPissimo issues
- `PULPissimo_pr.xlsx`: defect-related pull requests collected from PULPissimo
- `CV32E40P_issue.xlsx`: defect records collected from CV32E40P issues
- `CV32E40P_pr.xlsx`: defect-related pull requests collected from CV32E40P
- `Vivado_discussions.xlsx`: defect-related discussion records collected from Vivado community

## Description
This dataset is released to support replication and follow-up research on soft IP core defects.
